# Prompt: Generate "SFS Connect — Teacher Portal" PPT

Use this prompt with the `pptx` skill (or paste into a fresh Claude conversation) to generate
the teacher-facing training deck.

---

Create a PowerPoint presentation titled **"SFS Connect — Teacher Portal Guide"** for teachers
at St. Francis De Sales Secondary School, ahead of the app's launch. Audience: non-technical
teaching staff seeing this for the first time. Tone: simple, reassuring, step-by-step — not
a technical spec.

Cover these sections, one per slide (split into 2 slides if a section has more than ~5 bullet points):

1. **Title slide** — "SFS Connect — Teacher Portal", subtitle "A quick guide for teachers", school name.
2. **What is SFS Connect?** — one-line pitch: a single app to replace paper registers, diaries,
   and manual attendance/leave tracking.
3. **Logging in** — how to access the portal (web link / APK), first-time login.
4. **Dashboard** — the home screen overview teachers land on.
5. **My Students** — viewing class/subject student lists.
6. **Attendance** — marking class attendance; GPS-based self check-in/out for the teacher's own
   attendance.
7. **My Schedule** — viewing daily/weekly timetable.
8. **Homework** — assigning and tracking homework.
9. **Notices** — receiving school announcements.
10. **Exams** — viewing exam schedules.
11. **My Subjects & Entering Marks** — subjects/classes assigned to them by the admin, and how
    to enter marks for those subjects.
12. **Leave** — applying for leave through the app.
13. **My Profile** — updating personal details.
14. **Tips & Support** — who to contact if something doesn't work.
15. **Closing slide** — thank you / launch date placeholder.

Style: clean, modern, glassmorphic-friendly color palette (soft blues/greens, white cards) to
loosely match the app's own UI. Use icons/emoji sparingly for visual anchors per section
(e.g. 📋 Attendance, 📝 Homework, 📢 Notices). Keep bullet text short (under 12 words per line)
since this will be presented live, not read as a document.

Do not invent features not listed above. All listed features (including Enter Marks) are live
and should be described as available now, not "coming soon."
