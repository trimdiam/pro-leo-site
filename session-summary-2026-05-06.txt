1. FIX — House Points Visible on Homepage (Firestore Rules)
------------------------------------------------------------
Problem: The homepage house-points leaderboard showed empty skeleton boxes
for unauthenticated visitors because the `house_points` Firestore collection
was missing from the public-read rules and fell through to the auth-required
catch-all.

Fix: Added explicit public-read rules for:
  - /house_points/{docId}
  - /house_points/{houseId}/monthly_entries/{entryId}

File changed: firestore.rules


2. FEAT — Auto Monthly Summation for House Points (Admin Portal)
----------------------------------------------------------------
Added a month/year selector to the House Points admin section. When an admin
enters points for a selected house and month, the system:
  - Saves to subcollection: house_points/{name}/monthly_entries/{YYYY-MM}
  - Recalculates total from ALL monthly entries automatically
  - Updates house_points/{name} with totalPoints + currentMonthPoints

Admin panel also shows a monthly breakdown (last 6 months) per house.

Files changed: index.html (HTML selectors + JS functions:
  buildHouseMonthOptions, loadMonthlyEntry, saveHousePoints, loadAdminHousePoints)


3. FEAT — Priority Colour-Coded Notice Cards (Admin Portal)
-----------------------------------------------------------
Redesigned the School Notices admin section with modern card UI:
  - Urgent  → red card (#fff5f5, border #dc2626) with exclamation icon
  - Important → amber card (#fffbeb, border #d97706) with triangle-warning icon
  - Normal  → blue card (#eff6ff, border #3b82f6) with circle-info icon

Admin form includes a live preview dot and hint text that updates when priority
is changed. Each card has a delete button.

Files changed: index.html (HTML preview dot + JS loadAdminNotices,
  updatePriorityPreview), styles.css (notice card and badge CSS)


4. FEAT — Teacher Leave Ledger with Admin Quota Config
------------------------------------------------------
Teacher portal Leave section now shows a full ledger:
  - Summary card: Quota / Days Taken / Balance with progress bar
  - Balance turns amber when ≤ 3 days remain, red when 0 or overdrawn
  - History table: Date Range, Days, Type, Reason, Status (colour-coded rows)

Admin portal Leave section (new):
  - Quota configuration card: set annual leave days saved to
    settings/leave_config in Firestore
  - Applications table with Pending / Approved / Rejected / All filter
  - Approve and Reject action buttons for pending applications

Day calculation: Math.max(1, round((dateTo - dateFrom) / 86400000) + 1)

Files changed: index.html (HTML leave summary cards + admin a-leave section +
  JS functions: loadLeaveHistory, loadLeaveSummary, loadLeaveQuota,
  saveLeaveQuota, loadAdminLeave, adminApproveLeave, adminRejectLeave,
  _calcDays), styles.css (leave ledger CSS)
